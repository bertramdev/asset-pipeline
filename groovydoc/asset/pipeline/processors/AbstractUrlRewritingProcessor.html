Package: asset.pipeline.processors

[Groovy] Class AbstractUrlRewritingProcessor

      • asset.pipeline.processors.AbstractUrlRewritingProcessor

      • Subclasses iterate over relative asset URLs in a base asset file and rewrites the URL relative to the base asset file. In precompiler mode, the asset URLs are also cache digested.
