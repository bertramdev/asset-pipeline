Package asset.pipeline

  • Interface Summary 
    Interface Description
    AssetFile Provides an interface for referencing files resolved by the Asset-Pipeline AssetResolvers
    Processor Processor for an AssetFile
  • Class Summary 
    Class Description
    AbstractAssetFile This is the base Asset File specification class.
    AbstractProcessor Abstract implementation of the Processor interface
    AssetCompiler Build time compiler for assets.
    AssetHelper Helper class for resolving assets
    AssetPipeline Provides methods for fetching contents of assets within the asset-pipeline
    AssetPipelineConfigHolder Holder for Asset Pipeline's configuration Also Provides Helper methods for loading in config from properties
    AssetPipelineResponseBuilder
    AssetSpecLoader Loads asset specification.
    CacheManager A Cache Manager for the Asset-Pipeline runtime.
    CachePersister Receives asynchronous cache persistance requests and executes them.
    CssAssetFile An AssetFile implementation for CSS This currently applies a CssProcessor onto the css that does relative url replacement with digest named files
    DirectiveProcessor This processor is a base processor put in front of most asset processing.
    GenericAssetFile Generic implementation of the AssetFile interface This is used to reference objects that do not get otherwise processed.
    HtmlAssetFile An AssetFile implementation for Html This currently applies a HtmlProcessor onto the Html that does relative url replacement with digest named files
    JsAssetFile An AssetFile implementation for Javascript
